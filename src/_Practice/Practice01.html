== 문단 태그 ==

p : 문단 나눔

h : 글제목
br : 줄 바꿈
hr : 문단을 나눠 주는 용도로 수평선 줄 긋기

빨간색 글씨 사이에 파란색 글씨를 넣을 때는 span을 사용

abbr : HTML 축약어의 원래 단어 말풍선 표현
q와 blockquote 태그
q : 짧은 글 인용문을 인용부호로 표현
blockquote :
긴 글의 인용문을 들여쓰기로 표현하는 블록 요소
meter : 측정값 70% 그래프 표현 
pre요소는 문서 소스에서 보이는 그대로
여백과 줄 바꿈이 보 여 집 니 다.
